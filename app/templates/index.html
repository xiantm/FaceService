验证图片是否存在人脸

    POST -> http://ip:port/face/validate
    param file 需要对比的图片
    return: {"status": status,"message": message,'face_size': 人脸数}

1：1对比

    POST -> http://ip:port/face/compare/11
    param source 原图片
    param target 对比图片
    param tolerance 偏差
    return: {"status": True/False,"message": matching/mismatching}

添加人脸

    POST -> http://ip:port/face/add
    param file 需要上传的图片
    param name 姓名
    param group_id 组ID
return: {"status": true/false,"message": message}

验证人脸是否存在

    POST -> http://ip:port/face/exists
    param file 需要对比的图片
    param tolerance 偏差
    param group_id 组id
    return: {"status": status, "file_path": 图片地址, "name": 用户姓名, "message": message}

查找出所有包含人脸的照片

    POST -> http://ip:port/face/findall
    param file 需要对比的图片
    param tolerance 偏差
    param group_id 组id
    return: {"status": status, data:[]图片集合, "message": message}
